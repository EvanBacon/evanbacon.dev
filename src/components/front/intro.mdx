I'm Evan Bacon. I'm currently building [**Expo Agent**](https://agent.expo.dev), the agentic interface for building and deploying any app or website. I started working on the **Expo** project in 2017 with the goal to make mobile experiences easy to create, share, and discover. Previously at Expo, I created [**Expo Router**](https://expo.dev) (first file-based router for native apps), wrote Expo CLI, Expo for web, API routes, and deleted the `expo eject` command, replacing it with [Continuous Native Generation](https://docs.expo.dev/workflow/continuous-native-generation/).
